# pea-cli


> CLI tool for peajs

## Installation

```bash
npm i -g pea-cli
```

## Command

### `pea new`

Init a new pea project.

### `pea dev`

Runs the app in development mode.

Open http://localhost:3000 to view it in the browser.

### `pea test`

Runs the test watcher in an interactive mode.

### `pea build`

Builds the app for production to the build folder.
